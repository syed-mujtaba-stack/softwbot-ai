## Feature Request Process

### Submission

1. User submits feature request via dashboard
2. Product team reviews weekly
3. Votes counted from users
4. Prioritized in roadmap

### Evaluation Criteria

| Criterion | Weight |
|-----------|--------|
| User demand (votes) | 30% |
| Business impact | 25% |
| Technical feasibility | 20% |
| Strategic alignment | 15% |
| Effort/ROI | 10% |
